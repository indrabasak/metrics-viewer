Timer Metrics Viewer

A timer measures at which a piece of code is called and the distribution of its duration. A timer metrics viewer displays the following graphs:

  1. Duration graph displays the min, mean, and max metric values.
  2. Histogram graph displays the percentile metric values.
  3. Frequency graph displays the rate metric values.
A timer viewer is created by calling addTimer. 

metricsViewer.addTimer('#timer-div',
    "Timer Metrics Viewer Example",
    "Metric Viewer Example of Timer Metrics", "timer.test.metric.a");

A single property of a timer metric can be displayed in a graph by calling addTimerWithProperty and providing a property name along with a metric name. 

metricsViewer.addTimerWithProperty('#timer-prop-div',
    "Timer Metrics Viewer with One Property Example",
    "Metric Viewer Example of a Single Timer Metric Property", "timer.test.metric.a", "min");

There is no multiple timer metrics viewer.

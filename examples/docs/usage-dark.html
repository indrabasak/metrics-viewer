Dark Style

If you prefer darker look and feel, please add the metricsviewer-dark.css and metricsgraphics-dark.css to your style section. You still need the other style sheets. 

<html>
    <head>
        <title>Simple Metrics Viewer Page</title>

        <link href='css/metricsgraphics.css' rel='stylesheet' type='text/css'>
        <link href='css/metricsviewer.css' rel='stylesheet' type='text/css'>

        <link href='css/metricsgraphics-dark.css' rel='stylesheet' type='text/css'>
        <link href='css/metricsviewer-dark.css' rel='stylesheet' type='text/css'>

    </head>
    <body>
    </body>
</html>

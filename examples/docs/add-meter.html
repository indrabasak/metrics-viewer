Meter Metrics Viewer

A meter metric measures the rate of events over time. A meter metrics viewer displays four different values (1-min rate, 5-min rate, 15-min rate, and mean rate) in one graph. A meter metric viewer is created by calling addMeter. 

metricsViewer.addMeter('#meter-div',
     "Meter Metrics Viewer Example",
     "Metric Viewer Example of Meter Metrics", "meter.test.metric.a");

There is no multiple gauge metrics viewer.

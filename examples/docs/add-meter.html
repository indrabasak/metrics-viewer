Meter Metrics Viewer

A meter metric measures the rate of events over time. A meter metrics viewer displays four different values (1-min rate, 5-min rate, 15-min rate, and mean rate) in one graph. A meter metric viewer is created by calling addMeter. 

metricsViewer.addMeter('#meter-div',
     "Meter Metrics Viewer Example",
     "Metric Viewer Example of Meter Metrics", "meter.test.metric.a");

A single property of a meter metric can be displayed in a graph by calling addMeterWithProperty and providing a property name along with a metric name. 

metricsViewer.addMeterWithProperty('#meter-prop-div',
     "Meter Metrics Viewer with One Property Example",
     "Metric Viewer Example of a Single Meter Metric Property", "meter.test.metric.a", "m1_rate");

There is no multiple meter metrics viewer.

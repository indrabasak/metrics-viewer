Standard Usage

The bare minimum for using MetricsViewer.js on a web page is linking to the MetricsViewer.js, MetricsGraphics.js, D3, and jQuery libraries along with the styles of MetricsViewer.js and MetricsGraphics.js.

Create a HTML page and add the specified css and the Javascript files. Assuming the MetricsViewer.js and MetricsGraphics.js related css and JavaScript files are located in css and js folders respectively. 

<html>
    <head>
        <title>Simple Metrics Viewer Page</title>

        <link href='css/metricsgraphics.css' rel='stylesheet' type='text/css'>
        <link href='css/metricsviewer.css' rel='stylesheet' type='text/css'>

        <script src="https://d3js.org/d3.v4.min.js"></script>
        <script src='js/metricsgraphics.js'></script>
        <script src='js/metricsviewer.js'></script>
    </head>
    <body>
    </body>
</html>

Here is an example of adding, initializing, and refreshing the MetricsViewer.js.

<html>
    <head>
        /* add the style sheets and javascripts from above */

        <script>
        function addViewer() {
            metricsViewer.addCounter('#counter-div', "Counter Metrics Viewer Example",
            "Metric Viewer Example of Counter Metrics", "counter.test.metric.a");
        }

        $(document).ready(function () {
            addViewers();
            metricsViewer.init();
            keepRefreshingViewer();
        });

        function keepRefreshingViewer() {
            refreshViewer();
            setTimeout(refreshViewer, 1000);
        }

        //fetching metrics from an endpoint using JQuery ajax call
        function refreshViewer() {
            var url = "http://example.com";
            $.ajax({
                contentType: "application/json",
                url: url,
                success: function (data) {
                    metricsViewer.refresh(data);
                },
                error: function (xhr, ajaxOptions, thrownError) {
                    alert("Error - " + xhr.status + "Message: " + thrownError);
                },
                async: true
            });
        }
        </script>
    </head>
    <body>
        <div id="counter-div"></div>
    </body>
</html>

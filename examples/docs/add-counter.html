Counter Metrics Viewer

A counter metric is used to measure the count of an item you are trying to monitor, e.g., object instances, etc. It can be incremented or decremented. A counter metric viewer is created by calling the addCounter method. 

//add a new metrics viewer
metricsViewer.addCounter('#counter-div',  //div id
    "Counter Metrics Viewer Example", //title of the viewer
    "Metric Viewer Example of Counter Metrics", //description of the viewer
    "counter.test.metric.a"); //metric path

If you are interested in showing more than one counter metrics on the same viewer, call addMultiCounter and provide an array of metric names (instead of one metric name in the previous example). 

metricsViewer.addMultiCounter('#counter-multi-div',
    "Multiple Counter Metrics Viewer Example",
    "Metric Viewer Example of Multiple Counter Metrics",
    ["counter.test.metric.a", "counter.test.metric.b"]);

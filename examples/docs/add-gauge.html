Gauge Metrics Viewer

A gauge metric is similar to counter metrics and it is used to show an instantaneous measurement of a value. A gauge metric viewer is created by calling addGauge method. 

metricsViewer.addGauge('#gauge-div',
    "Gauge Metrics Viewer Example",
    "Metric Viewer Example of Gauge Metrics", "gauge.test.metric.a");

Multiple gauge metrics can be displayed in a single graph by calling addMultiGauge and providing an array of metric names. 

metricsViewer.addMultiGauge('#gauge-multi-div',
    "Gauge Metrics Viewer Example",
    "Metric Viewer Example of Gauge Metrics",
    ["gauge.test.metric.a", "gauge.test.metric.b"]);

Getting Started

MetricsViewer.js is a simple JavaScript library for displaying Dropwizard Metrics data as line graphs. Dropwizard metrics are formerly known as Coda Hale/Yammer metrics. The MetricsViewer.js library is based on MetricsGraphics.js charting library. This library is very easy to use.

Adding a Graph

Before you do anything with the metrics viewer, you have to create a graph by calling one of the add methods of the metricsViewer. The type of add method is based on the type of metric you are trying to display. The addadd method takes following the parameters:

  • Id of the HTML division tag where you want the metric graph to be displayed
  • Title of the graph
  • Description of the graph
  • Qualified name of the metric

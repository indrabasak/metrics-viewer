JVM Metrics Viewer

A metric viewer to display metrics related to JVM. A JVM metrics viewer displays the following graphs:

  1. Memory usage
  2. Heap usage
  3. Eden space usage
  4. Survivor space usage
  5. Old Gen usage
  6. Thread usage
A JVM metric viewer is created by calling addJvm. 

metricsViewer.addJvm('#jvm-div',
    "JVM Metrics Viewer Example",
    "Metric Viewer Example of JVM Metrics");

There is no multiple JVM metrics viewer.
